A part of the problem was clients were not supplying web content in time for a scheduled build to commence leading to missed deadlines.

I built a content check-in system based on Wordpress, allowing clients to easily submit copy, images, meta-content if necessary, and supply instructions. The system provided an always-up-to-date report on the status of content-supply. Our project managers could now plan suitably.

There were a number of other features built-in like page-locking & downloading of packages of content.
